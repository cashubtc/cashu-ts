    Class ReceiveBuilder

    Builder for receiving a token.

    Remarks

    If you do not call a type method, the wallet’s policy default is used.

    Example

    const proofs = await wallet.ops
        	.receive(token)
        	.asDeterministic() // counter 0 auto reserves
        	.requireDleq(true)
        	.run();
