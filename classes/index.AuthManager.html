    Class AuthManager

    AuthManager.

    • Owns CAT lifecycle (stores, optional refresh via attached OIDCAuth)
    • Mints and serves BATs (NUT-22)
    • Validates DLEQs for BATs per NUT-12.
    • Supplies serialized BATs for 'Blind-auth' and CAT for 'Clear-auth'
