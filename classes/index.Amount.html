    Class Amount

    Immutable, non-negative integer amount value object.

    Internal representation is bigint. Use factory methods to instantiate.

    Example

    Amount.from('21'); // string
        Amount.from(21); // number
        Amount.from(21n); // bigint
        Amount.zero();
        Amount.one();
